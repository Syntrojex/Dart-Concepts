# 📖 Introduction

Before writing Dart syntax and building Flutter applications, it is
important to understand the ecosystem in which these technologies exist.

This section builds the foundation for the journey ahead — from the
mobile platforms Dart and Flutter target, to the application formats used
to distribute apps, and the concepts that connect Dart with Flutter
development.

The goal is simple: understand the **"why" and "where"** before learning
the **"how"**.
